# Sorteio de Fóruns AVA

Extensão Chrome para realizar sorteios de autores dentro de fóruns do AVA.

Extensão desenvolvida para a diciplina de Resolução de Problemas Estruturados II.

---

### Visão geral

Esta extensão adiciona um painel dentro de páginas de fórum do AVA. A partir desse painel é possível coletar autores das discussões e gerar grupos automaticamente, como meio a meio, 3 grupos, 4 grupos ou duplas.

---

### Estrutura de arquivos

- manifest.json
- background.js
- estilos.css
- content.js
- generate_chrome_icon.py
- icons/

---

### Instalação

1. Abra chrome://extensions/

2. Ative o modo desenvolvedor

3. Clique em "Carregar sem compactação"

4. Selecione a pasta da extensão
